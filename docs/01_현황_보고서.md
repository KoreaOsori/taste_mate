# [보고서 01] Ricefreind 프로젝트 현황 보고서 (Part 1 & 2 요약)

본 문서는 **Ricefreind (식구)** 프로젝트의 현재 진행 상황을 팀원들과 공유하고, 협업 환경으로의 전환을 돕기 위해 작성되었습니다.

## 🚀 기술 스택 개요

- **프론트엔드**: React 18 + Vite + Tailwind CSS
- **백엔드**: FastAPI (Python 3.10+)
- **인프라**: Supabase (인증, 데이터베이스, Edge Functions)

## 🛠️ 연동 API 현황

| API | 목적 | 상태 |
| :--- | :--- | :--- |
| **카카오 로그인** | 소셜 로그인 연동 | `.env`에 키 설정 완료 |
| **카카오 맵** | 식당 위치 및 지도 표시 | `.env`에 키 설정 완료 |
| **네이버 검색** | 주변 식당 검색 서비스 | 설정 완료 |
| **OpenWeather** | 날씨 기반 음식 추천 | 설정 완료 |
| **식품영양성분 API** | 음식별 영양 데이터 제공 | 설정 완료 |

## 📂 프로젝트 구조

### 백엔드 (`/backend`)
- `main.py`: 엔트리 포인트, 라우터 설정.
- `api/endpoints/`:
  - `recommend.py`: AI 추천 로직.
  - `profile.py`: 사용자 프로필 관리.
  - `meals.py`: 식사 기록 (Mock Data 활용 중).

### 프론트엔드 (`/frontend`)
- `src/App.tsx`: 메인 라우팅 및 화면 전환.
- `src/components/`: 모듈화된 UI 컴포넌트.

## ⚙️ 로컬 환경 설정 가이드

1. **의존성 설치**:
   ```bash
   cd frontend && npm install
   cd ../backend && pip install -r requirements.txt
   ```
2. **환경 변수**: `.env.example`을 `.env`로 복사하여 사용.
